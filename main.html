Research Proposal Flowchart

A high-level diagram illustrating the research design for "Emergent Misalignment in Debate-Based Multi-Agent Systems."

1. Agent Configuration

Base LLM

Foundation for all agents

LLaMA-7B or Mistral-7B

Debater A (Aligned) & Manager
Prompted for fairness or fine-tuned on balanced datasets like Debatepedia.
Debater B (Misaligned)
Fine-tuned on misalignment-inducing data like Jigsaw Toxic Comments.

2. Debate Simulation

Debate Scenario
A social media post or comment requiring a moderation decision.
Structured Debate
Debater A

Presents aligned argument

vs
Debater B

Presents misaligned argument

Manager Agent's Role
Observes the debate and outputs a final Moderation Decision & Rationale.

3. Evaluation & Analysis

A. Direct Influence Analysis
Analyze the Moderation Decision & Rationale to see if the Manager Agent was influenced by the misaligned debater.
B. Emergent Misalignment Testing

The Manager Agent (post-debate) is tested on a suite of Unrelated, Cross-Domain Tasks.

  • Policy Analysis
  • Negotiation Advice
  • Historical Interpretation
  • Ethical Q&A

Performance is measured using benchmarks:

TruthfulQA Machiavelli
Final Comparison
The results from the experimental group are compared against a Control Group where all agents were aligned.
